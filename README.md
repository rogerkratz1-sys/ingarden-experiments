ingarden-experiments v1.1.4

Citation
--------
Roger Kratz. 2026. ingarden-experiments v1.1.4. Zenodo. https://doi.org/10.5281/zenodo.19011284

Summary
-------
This release bundles the project source, documentation, and two distribution archives for reproducible experiments and example data. Use the full archive for complete data and code; use the no-data archive for code only.

Included files
--------------
- ingarden-v1.1.4_full.zip
- ingarden-v1.1.4_no-data.zip

Usage
-----
1. Download the desired ZIP file from the Zenodo record.
2. Extract to a working directory.
3. Follow README.md inside the archive for installation and example runs.
