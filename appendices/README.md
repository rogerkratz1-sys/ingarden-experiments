# Appendices Directory

This directory contains the canonical versions of Appendix A through Appendix I. Each appendix is stored as a standalone text file and corresponds to long-form methodological, analytical, or interpretive material referenced in the manuscript.

## Canonical Appendices

- **Appendix A** — appendices/Appendix A.txt  
- **Appendix B** — appendices/Appendix B.txt  
- **Appendix C** — appendices/Appendix C.txt  
- **Appendix D** — appendices/Appendix D.txt  
- **Appendix E** — appendices/Appendix E.txt  
- **Appendix F** — appendices/Appendix F.txt  
- **Appendix G** — appendices/Appendix G.txt  
- **Appendix H** — appendices/Appendix H.txt  
- **Appendix I** — appendices/Appendix I.txt  

These files are the authoritative appendix materials referenced by the manuscript.

## Structure and Conventions

- All appendices reside exclusively in this directory.  
- No appendix content should appear in `docs/` or the repository root.  
- Appendices complement the supplements by providing extended analyses, methodological details, and supporting materials that do not belong in the main text.

## Notes

- The repository root contains a `manifest.txt` indexing all supplements, appendices, and artifact bundles.  
- The `docs/` directory contains only pointer files for supplements and does not store appendix content.
