Ingarden Experiments
===================

**Short citation**
Roger Kratz. 2026. ingarden‑experiments v1.1.4. Zenodo. https://doi.org/10.5281/zenodo.19011284

Purpose
-------
This repository is the canonical home for the Ingarden Project code, data, analyses, and reproducibility materials. It is intended for researchers, reviewers, and anyone who needs a fully auditable, archival copy of the project.

Quick start
-----------
- For full reproducibility and example data, download **ingarden-v1.1.4_full.zip** from the Zenodo record.
- For code only, download **ingarden-v1.1.4_no-data.zip**.
- See `README.md` for installation and example runs.

What this repository contains
-----------------------------
- Core scripts and analysis pipelines
- Experimental workflows and diagnostics
- Reproducibility materials and supplement bundles
- Provenance notes and configuration files
- Artifacts prepared for archival release (Zenodo)

Verification and citation
-------------------------
- Zenodo DOI: https://doi.org/10.5281/zenodo.19011284
- Checksums (SHA256) are included in `CHECKSUMS_SHA256.txt` in the release and in the Zenodo deposit.
- Please cite the DOI when using or referencing this work.

Structure
---------
A typical layout for this repository is:
- `src/` analysis code and scripts
- `notebooks/` example notebooks and demonstrations
- `data/` small example datasets (large data are in the Zenodo archive)
- `docs/` documentation and usage notes
- `supplements/` reproducibility bundles and manifests
