5. Notes on Repository Structure
-------------------------------------------------------------------------------

- All canonical supplement texts reside exclusively in the supplement/ directory.
- All appendices reside exclusively in the appendices/ directory.
- The docs/ directory contains only lightweight pointer files for Supplements S-1 through S-8.
- Artifact bundles for S-7 and S-8 contain reproducibility materials, diagnostics, mappings, and results.
- No long-form materials should be stored directly in docs/ or the repository root.
