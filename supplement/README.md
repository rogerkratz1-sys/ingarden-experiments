# Supplement Directory

This directory contains the canonical versions of all eight supplements associated with the manuscript. Each supplement is stored as a standalone text file. Supplements S‑7 and S‑8 include additional artifact bundles that provide reproducibility materials, diagnostics, and configuration files.

## Canonical Supplements

- **Supplement S‑1** — supplement/Supplement S-1.txt  
- **Supplement S‑2** — supplement/Supplement S-2.txt  
- **Supplement S‑3** — supplement/Supplement S-3.txt  
- **Supplement S‑4** — supplement/Supplement S-4.txt  
- **Supplement S‑5** — supplement/Supplement S-5.txt  
- **Supplement S‑6** — supplement/Supplement S-6.txt  
- **Supplement S‑7** — supplement/Supplement S-7.txt  
- **Supplement S‑8** — supplement/Supplement S-8.txt  

These files are the authoritative versions referenced by the manuscript.

## Artifact Bundles

### S‑7 Artifact Bundle  
Located in `supplement/S-7/`.  
Contents include:
- README_S7.txt  
- Diagnostics and stability outputs  
- Eventizations, mappings, and intermediate results  
- Checksums and provenance files  

### S‑8 Artifact Bundle  
Located in `supplement/S-8/`.  
Contents include:
- S8_readme.txt  
- Stability maps and diagnostics  
- Null summaries and configuration files  
- Additional reproducibility materials  

## Pointer Files

The `docs/` directory contains lightweight pointer files for Supplements S‑1 through S‑8. These exist to preserve legacy paths and ensure manuscript links remain stable. All canonical content resides here in the `supplement/` directory.

## Notes

- No supplement text should appear in `docs/`.  
- All reproducibility materials belong in the S‑7 and S‑8 artifact bundles.  
- This directory is the single source of truth for all supplement content.
